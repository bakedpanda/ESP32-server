    FUNCTION: check_board_space(port: str) -> dict
    - Run: subprocess.run(["mpremote", "connect", port, "df"], capture_output=True, text=True, timeout=15)
    - If returncode != 0: return {"error": "board_unreachable", "detail": result.stderr.strip()}
    - Parse stdout for filesystem usage. mpremote df outputs lines like:
        /       : 512000 bytes total,  200000 bytes used,  312000 bytes free
    - Extract used_bytes and total_bytes from the "/" filesystem line.
    - pct = int(used_bytes / total_bytes * 100)
    - if pct >= SPACE_FAIL_PCT: return {"error": "insufficient_space", "detail": f"Filesystem {pct}% full (limit {SPACE_FAIL_PCT}%)"}
    - if pct >= SPACE_WARN_PCT: return {"warning": "filesystem_70pct_full", "pct": pct}
    - return {"ok": True, "pct": pct}

    FUNCTION: verify_file_size(port: str, remote_path: str, local_size: int) -> dict
    - Run mpremote exec to get file size:
        subprocess.run(["mpremote", "connect", port, "exec",
                        f"import os; print(os.stat('{remote_path}')[6])"],
                       capture_output=True, text=True, timeout=10)
    - Parse int from stdout.strip()
    - If remote_size != local_size: return {"error": "file_integrity_failed", "detail": f"Expected {local_size} bytes, got {remote_size}"}
    - return {"ok": True}

    FUNCTION: deploy_file(port: str, local_path: str, remote_path: str | None = None) -> dict
    - remote_path defaults to pathlib.Path(local_path).name (filename only, board root)
    - Pre-flight space check: call check_board_space(port); if "error" in result: return it
    - Run: subprocess.run(["mpremote", "connect", port, "cp", local_path, f":{remote_path}"],
                          capture_output=True, text=True, timeout=30)
    - If returncode != 0: return {"error": "deploy_failed", "detail": result.stderr.strip()}
    - Integrity check: call verify_file_size(port, remote_path, pathlib.Path(local_path).stat().st_size)
    - If "error" in integrity result: return integrity result
    - Return dict (including any warning from space check):
        base = {"port": port, "files_written": [remote_path]}
        if "warning" in space_result: base["warning"] = space_result["warning"]
        return base

    FUNCTION: deploy_directory(port: str, local_dir: str) -> dict
    - Collect all files from local_dir recursively using pathlib.Path(local_dir).rglob("*")
    - Filter: skip if any part of path is in DEPLOY_EXCLUDE_DIRS, skip if suffix in DEPLOY_EXCLUDE_EXTS
    - Keep: .py, .json, .txt, and any other non-excluded files
    - Pre-flight space check once (before first file): if "error" in check: return it; store warning if present
    - For each file: call subprocess.run to mpremote cp, then verify_file_size
    - If any file fails: return {"error": "deploy_failed", "file": str(remote_path), "detail": ...}
    - Return: {"port": port, "files_written": [list of remote paths], optionally "warning": ...}

    Remote path for directory deploy: strip local_dir prefix, use remainder as relative path on board.
    e.g. local_dir="/home/user/myproject", file="/home/user/myproject/lib/utils.py" -> remote=":lib/utils.py"

    All functions follow Phase 1 error-dict pattern:
    - Never raise to callers
    - Always return {"error": "snake_case_code", "detail": "human string"} on failure
    - Always return a dict with meaningful keys on success
